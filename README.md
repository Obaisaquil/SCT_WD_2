# ⏱️ Stopwatch Web Application

## Overview

The **Stopwatch Web Application** is a responsive and interactive web-based timer developed using **HTML, CSS, and JavaScript**. It enables users to accurately measure time intervals with essential stopwatch functionalities, including **Start**, **Pause**, **Reset**, and **Lap Time Recording**.

This project demonstrates JavaScript timing functions, DOM manipulation, and responsive web design to create a smooth and user-friendly experience.

## Features

* ▶️ Start the stopwatch.
* ⏸️ Pause and resume the timer.
* 🔄 Reset the stopwatch and clear all lap times.
* 🏁 Record and display unlimited lap times.
* 📱 Responsive design for desktop, tablet, and mobile devices.
* 🎨 Clean and modern user interface.
* ⚡ Accurate time tracking with millisecond precision.

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open the `index.html` file in any modern web browser.
4. Use the buttons to control the stopwatch:

   * **Start** – Starts or resumes the stopwatch.
   * **Pause** – Pauses the current timer.
   * **Reset** – Resets the timer and clears lap records.
   * **Lap** – Saves the current elapsed time without stopping the stopwatch.

## Functionality

### Start

Begins timing from `00:00:00.00` or resumes from the previously paused time.

### Pause

Temporarily stops the stopwatch while preserving the elapsed time.

### Reset

Stops the stopwatch, resets the timer to zero, and removes all recorded lap times.

### Lap

Records the current elapsed time and displays it in the lap history, allowing users to compare multiple intervals.
